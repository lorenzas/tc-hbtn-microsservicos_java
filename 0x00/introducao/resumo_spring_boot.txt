Spring Boot Framework é o mais popular Java Framework para desenvolvimento de microsserviços. Entre seus principais benefícios e recursos, pode-se citar o fornecimento de as configurações básicas, um servidor de aplicação embarcado (Tomcat) e dependências necessárias para desenvolver uma aplicação, sem precisar configurar tudo do zero.
Esse framework utiliza o Apache Maven para baixar as dependências e sub-dependências (JPA, Hibernate, etc…) do seu projeto atráves do POM.xml. EM relação às IDEs, na minha opinião, a melhor para o desenvolvimento é o Intellij por causa de seu excelente suporte. Ainda, há o Spring Initializr, que é uma ferramenta baseada na Web que gera a estrutura do projeto Spring Boot.
